## 📌 Sobre o Projeto

Landing page desenvolvida para uma profissional da área médica, com o objetivo de apresentar seus serviços, transmitir credibilidade e facilitar o contato com pacientes de forma simples e eficiente.

---

## 🎨 Design e Experiência

O layout foi pensado para transmitir profissionalismo, confiança e clareza. A interface é limpa, organizada e focada na experiência do usuário, priorizando a fácil navegação e acesso às informações essenciais.

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 🚀 Funcionalidades

- Apresentação de serviços médicos  
- Seção de contato rápido 
- Layout moderno e intuitivo  
- Design responsivo  

---

## 📱 Responsividade

A página foi desenvolvida com abordagem responsiva, garantindo uma ótima experiência em dispositivos móveis, tablets e desktops.
